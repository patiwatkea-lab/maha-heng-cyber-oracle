---

## 🔮 ฟีเจอร์เด่น (Key Features)

* **👤 ข้อมูลพื้นดวงรายวัน:** วิเคราะห์ธาตุเจ้าเรือน, สีมงคล, สีกาลกิณี และทิศรับทรัพย์ตามวันเกิดจริง
* **🎲 ระบบคำนวณเลขมงคล:** ใช้ระบบ Seed-Calculation ประมวลผลร่วมกับวันเกิดและ "เลขจิตสัมผัส" เพื่อหาเลขที่ถูกโฉลกที่สุด
* **✍️ ถอดรหัสนิมิต:** ช่องบันทึกความฝันเพื่อนำพลังงานจากนิมิตมาคำนวณร่วมกับฐานดวง
* **📊 กราฟใยแมงมุม (Fate Radar):** แสดงระดับพลังงานใน 5 ด้าน: โชคลาภ, อำนาจ, การเงิน, สุขภาพ และเสน่ห์

---

## 💻 วิธีการรันโปรแกรม (For Developers)

หากต้องการรันโปรแกรมนี้บนเครื่องคอมพิวเตอร์ของคุณเอง ให้ทำตามขั้นตอนดังนี้:

1. **ติดตั้ง Python:** ตรวจสอบว่าเครื่องของคุณมี Python 3.9 ขึ้นไป
2. **ดาวน์โหลดไฟล์:** ดาวน์โหลดไฟล์ทั้งหมดจาก Repository นี้ไปไว้ในเครื่อง
3. **ติดตั้ง Library ที่จำเป็น:** เปิด Terminal/Command Prompt ในโฟลเดอร์โปรเจคแล้วพิมพ์:
   ```bash
   pip install -r requirements.txt
4. **เริ่มใช้งานแอป:** พิมพ์คำสั่งด้านล่างเพื่อรันหน้าเว็บ:
   ```bash
   streamlit run app.py
   
   ## 🛡️ รายละเอียดการวิเคราะห์ชะตา (Fate Analysis Details)

ระบบใช้การคำนวณเชิงอัลกอริทึมร่วมกับโหราศาสตร์เพื่อแยกแยะพลังงานดังนี้:

| หัวข้อการวิเคราะห์ | รายละเอียดและความหมาย |
| :--- | :--- |
| 🍀 **โชคลาภ (Luck)** | โอกาสที่จะได้รับสิ่งดีๆ โดยไม่คาดฝันในรอบวัน |
| ⚡ **อำนาจ (Power)** | บารมีและการได้รับการยอมรับในการเจรจาหรือสั่งการ |
| 💰 **การเงิน (Wealth)** | สภาพคล่องและการดึงดูดทรัพย์สินตามธาตุเจ้าเรือน |
| ❤️ **สุขภาพ (Health)** | พลังกายและพลังใจในการต่อสู้กับอุปสรรค |
| ✨ **เสน่ห์ (Charm)** | ความเมตตามหานิยมและการได้รับความช่วยเหลือจากผู้อื่น |

## 🎨 การออกแบบ (Design Concept)

โปรเจคนี้ถูกออกแบบภายใต้คอนเซปต์ **"Midnight Serenity"** โดยเน้น UI ที่ดูลึกลับแต่ทันสมัย:

* 🔵 **Neon Glow:** ใช้สี Cyan และ Purple เพื่อสร้างอารมณ์ Sci-Fi ล้ำยุค
* 🌑 **Dark Mode:** พื้นหลังดำสนิทเพื่อลดการใช้สายตาและเน้นความขลังของข้อมูล
* 📱 **Responsive:** รองรับการใช้งานที่สมบูรณ์แบบทั้งบน Desktop และ Mobile
