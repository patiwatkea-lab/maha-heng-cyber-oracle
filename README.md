---

## 🔮 ฟีเจอร์เด่น (Key Features)

* **👤 ข้อมูลพื้นดวงรายวัน:** วิเคราะห์ธาตุเจ้าเรือน, สีมงคล, สีกาลกิณี และทิศรับทรัพย์ตามวันเกิดจริง
* **🎲 ระบบคำนวณเลขมงคล:** ใช้ระบบ Seed-Calculation ประมวลผลร่วมกับวันเกิดและ "เลขจิตสัมผัส" เพื่อหาเลขที่ถูกโฉลกที่สุด
* **✍️ ถอดรหัสนิมิต:** ช่องบันทึกความฝันเพื่อนำพลังงานจากนิมิตมาคำนวณร่วมกับฐานดวง
* **📊 กราฟใยแมงมุม (Fate Radar):** แสดงระดับพลังงานใน 5 ด้าน: โชคลาภ, อำนาจ, การเงิน, สุขภาพ และเสน่ห์

## 💻 วิธีการรันโปรแกรม (For Developers)

หากต้องการรันโปรแกรมนี้บนเครื่องคอมพิวเตอร์ของคุณเอง ให้ทำตามขั้นตอนดังนี้:

1. **ติดตั้ง Python:** ตรวจสอบว่าเครื่องของคุณมี Python 3.9 ขึ้นไป
2. **ดาวน์โหลดไฟล์:** ดาวน์โหลดไฟล์ทั้งหมดจาก Repository นี้ไปไว้ในเครื่อง
3. **ติดตั้ง Library ที่จำเป็น:** เปิด Terminal/Command Prompt ในโฟลเดอร์โปรเจคแล้วพิมพ์:
   ```bash
   pip install -r requirements.txt
4. เริ่มใช้งานแอป: พิมพ์คำสั่งด้านล่างเพื่อรันหน้าเว็บ 
      Bash
   streamlit run app.py
   
📖 วิธีการใช้งานแอป (User Guide)
1.ระบุตัวตน: กรอกชื่อของคุณและเลือกวันเกิดในช่องปฏิทิน ระบบจะแสดงพื้นดวงชะตาทันทีในกรอบด้านซ้าย

2.บันทึกนิมิต: หากมีความฝันหรือนิมิต สามารถกรอกลงในช่อง "บันทึกนิมิต/ความฝัน" เพื่อใช้เป็นปัจจัยในการคำนวณเลข

3.ใส่เลขจิตสัมผัส: ระบุตัวเลข 0-999 ที่คุณรู้สึกถึงพลังงานในขณะนั้น

4.วิเคราะห์ดวง: กดปุ่ม [เริ่มการวิเคราะห์ดวงชะตา] เพื่อรับเลขมงคล 2 หลัก พร้อมกราฟวิเคราะห์พลังงานชีวิต

5.ตรวจสอบประวัติ: ดูสถิติเลขที่เคยสุ่มได้ที่แถบ "ประวัติการวิเคราะห์" ทางด้านขวา
